Homerules and Expectations


This is where you'll find the various rules for the campaign that might not be standard for core D&D. This might include various optional rules we'll be implementing, general rules regarding what is or isn't allowed during roleplay (because some people have boundaries), and other changes from core D&D rules (like adjustments to classes/features that I don't like lol). This section also includes more 4th-wall-breaking expectations, including establishing cancellation policies and whatnot. In other words, this is all the boring housekeeping stuff that tbh is probably unnecessary, but might get included for future reference.
